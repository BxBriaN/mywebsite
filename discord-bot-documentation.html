Custom Discord Bot Documentation

Introduction

This documentation provides an overview of the custom Discord bot I developed. The bot enhances user engagement and provides utility within Discord servers by offering features such as rotating status messages and a /sb command for sending images.

Features

  • Rotating Status Messages: The bot changes its status message periodically to keep the server environment dynamic and engaging.
  • /sb Command: Users can use this command to receive a random image or a specific image based on parameters.

Code Overview

Rotating Status Messages

The following code demonstrates how the bot rotates its status messages:


import discord
from discord.ext import tasks, commands

bot = commands.Bot(command_prefix='!')

status_messages = ["Hello World!", "Bot at your service.", "Securing the server."]

@tasks.loop(minutes=60)
async def change_status():
    await bot.change_presence(activity=discord.Game(next(status_cycle)))

status_cycle = iter(status_messages)

@bot.event
async def on_ready():
    change_status.start()
    print(f'Logged in as {bot.user}')

bot.run('YOUR_BOT_TOKEN')

/sb Command Implementation

This command sends an image when a user types /sb in the chat.


@bot.command()
async def sb(ctx):
    image_path = 'path/to/your/image.jpg'
    await ctx.send(file=discord.File(image_path))

Usage

Rotating Status Messages

The bot automatically rotates its status messages every 5 seconds. No user intervention is required.

/sb Command

To use the /sb command, simply type:

/sb

The bot will respond by sending the configured image.
